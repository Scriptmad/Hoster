## About Tool:
<i>This script is for hosting your own webiste using ngrok server.</i>

## Available On:

* TERMUX

### How to install and run( Termux ):

* `pkg install git`
* `git clone https://github.com/Scriptmad/Hoster`
* `cd Hoster`
* `bash Hoster.sh`
* `bash ngserver.sh`
